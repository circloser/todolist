# 팀 진행 체크리스트

팀원들이 프로젝트 진행 상황을 함께 확인하는 웹사이트입니다. 습지복원팀의
모든 업무를 표 형태로 놓고, 업무별 진행 단계가 가로 방향으로 표시됩니다.
담당자 필터, 정렬, 드래그 순서 변경, 하단 빠른 업무 추가를 지원합니다.

기본 진행 템플릿:

1. 계획(초안)
2. 견적
3. 계획 결재
4. 구매 요청
5. 계약 체결
6. 착수
7. 과업 진행(25%)
8. 과업 진행(50%)
9. 과업 진행(75%)
10. 과업 진행(100%)
11. 완료계 접수
12. 검수
13. 대금 지급 요청
14. 결과 보고

## 실행

```powershell
npm.cmd install
npm.cmd run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 빌드

```powershell
npm.cmd run build
```
